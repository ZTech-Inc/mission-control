If the docs route files do not exist yet, create them. If they exist with mock data, replace.

**Both routes follow the same pattern. Implement for teams first, then departments.**

**GET handler (both routes):**
1. `requireRole(request, 'viewer')` for auth.
2. Look up the entity (team or department) by ID from the database to get its name and department info.
3. Compute the `docsDir`:
   - Teams: `path.join(config.agentsDir, deptName, teamName, 'docs')`
   - Departments: `path.join(config.agentsDir, deptName, 'docs')`
4. If `!config.agentsDir` return 400 `{ error: "Agents directory not configured" }`.
5. If the docs directory does not exist (`!existsSync(docsDir)`), return `{ docs: [], content: {} }`.
6. Recursively read the directory to build `DocFile[]` tree:
   ```typescript
   import { readdirSync, readFileSync, statSync, existsSync } from 'node:fs'
   function scanDocs(dir: string, basePath: string = ''): DocFile[] {
     const entries = readdirSync(dir, { withFileTypes: true })
     return entries.map(entry => {
       const entryPath = path.join(basePath, entry.name)
       if (entry.isDirectory()) {
         return { name: entry.name, path: entryPath, type: 'directory' as const, children: scanDocs(path.join(dir, entry.name), entryPath) }
       }
       return { name: entry.name, path: entryPath, type: 'file' as const }
     }).sort((a, b) => (a.type === 'directory' ? -1 : 1) - (b.type === 'directory' ? -1 : 1) || a.name.localeCompare(b.name))
   }
   ```
7. Also read content for each file: build a `content: Record<string, string>` map reading each `.md` file.
8. Return `{ docs: scanDocs(docsDir), content }`.

**POST handler (both routes):**
1. `requireRole(request, 'operator')` for auth.
2. Validate body with zod: `{ filename: z.string().min(1).max(200), content: z.string().optional().default('') }`.
3. Sanitize filename: strip path separators, ensure `.md` extension if not present.
4. Compute `docsDir` same as GET.
5. `mkdirSync(docsDir, { recursive: true })`.
6. `writeFileSync(path.join(docsDir, filename), body.content, 'utf-8')`.
7. Return 201 `{ ok: true, path: filename }`.

**For teams route**: Look up team by ID from DB: `db.prepare('SELECT t.name, d.name as dept_name FROM teams t JOIN departments d ON t.department_id = d.id WHERE t.id = ?').get(params.id)`. Note: the actual column/join structure depends on the schema — read the existing route at `src/app/api/teams/[id]/route.ts` to see how team lookup works.

**For departments route**: Look up dept by ID: `db.prepare('SELECT name FROM departments WHERE id = ?').get(params.id)`.
